WELCOME TO THE HANDBALLSTATS HOMEPAGE!

This website visualizes the results of a project of mine, where I automatically download (scrape) and analyze data from the handball.ch matchcenter. I use Python (Selenium Webdriver, Pandas, Matplotlib) to collect the data and visualize it statically, as well as JavaScript (D3.js) to create the interactive visualizations on this website.

In the highest leagues of Swiss Handball, games (and player stats) are recorded using a feature called 'Liveticker'. Since I coach a team in such a league, I figured I would use this data, visualize it interactively and maybe gain some more insights into the performance of my team. Sadly, I could not find an API to get the data directly, so I had to resort to scraping it. Along the way I implemented a mechanism for data collection that lets me analyze additional teams with minimal effort, so therefore I ended up analyzing more than just my team's performance.

Here's a brief overview of the site and what to expect where:

Individual Player Statistics

Here you can find statistics of individual players. You can display all players on top of each other or select only some players to compare against each other.

Game Progressions || Team Statistics

Here you will find neat visualizations of a game's progression. Be sure to read the section about how these graphs are generated in the About Section to fully understand and be able to interpret them correctly.
